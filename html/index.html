Leitor de Faturas

Esse Projeto feito em Python para automatizar o processo de controle de gastos e leitura de faturas. Ele utiliza a biblioteca PyPDF2 para extrair texto de arquivos PDF, permitindo que o usuário busque por palavras-chave específicas, como "Cosumo do mês" ou "Valor a ser pago", e obtenha informações detalhadas sobre os gastos.

Atualização dos Dados do Power BI Automática

Esse Projeto foi uma automação feita pelo "Power Automate", foi uma forma simples e temporária para realizar essa coleta de dados, onde o Power Automate coleta os dados do Excel e atualiza automaticamente o Power BI. O fluxo é acionado diariamente de maneira manual, garantindo que os dados estejam sempre atualizados.
